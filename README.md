# 素晴らしいdotfiles
## これは何
聖なるdotfilesです。なんとこのたびansibleでインストール出来るようになりました。

## 対応ディストリビューション
- Ubuntu
- macOS
- ArchLinux

サーバー向けは意識していません。どうしても使いたい場合は気合い出してください。

## 使い方
### macOS
#### Prerequisites
- brew
- ansible

#### Apply
```bash
$ make apply-mac
```

### Ubuntu
#### Prerequisites
- ansible

#### Apply
```bash
$ make apply-ubuntu
```

### ArchLinux
#### Prerequisites
- ansible
- yay

#### Apply
```bash
$ make apply-archlinux
```
